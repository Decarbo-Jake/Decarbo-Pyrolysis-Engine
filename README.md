# Decarbo Pyrolysis Engine

Internal calculation engine for pyrolysis plant design, 
mass/energy balance, hardware sizing, and EU WID compliance.

## Structure
- `engine/`        — calculation modules (Modules 1–7)
- `client_model/`  — generates locked Excel client deliverables
- `tests/`         — validation tests against reference projects
- `projects/`      — per-project input data
- `docs/`          — methodology documentation

## First validated project: Jibito (BR-01 Broza de Caña)
